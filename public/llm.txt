## About
Sednium Oorty is a multi-model AI chat and coding orchestration platform. It provides a unified interface to interact with multiple AI providers, run local models, and orchestrate multi-agent workflows.

## Core Features
1. Multi-Model Orchestration — Powered by the proprietary Sednium Rosette API for intelligent routing and combination of multiple AI models
2. Multi-Provider Support — Connect to Google Gemini, OpenAI, Anthropic, xAI, Groq, OpenRouter, NVIDIA NIM, Ollama, and browser-native models
3. Specialized Chat Modes — Quick Mode (speed), Thinking Mode (deep reasoning), Coding Mode (software architecture)
4. Custom System Instructions — Fine-tune AI behavior on top of any Chat Mode
5. Configuration Presets — Save and switch between provider + model + mode + instructions combinations
6. Attachments & Context — Add files and images directly into chat context
7. Chat History Management — Local storage with pinning, renaming, and deletion
8. Local AI — Run open-source models via Ollama and LM Studio without internet
9. Hugging Face Integration — Download, cache, and run models from the Hugging Face hub
10. Privacy-First — API keys stored locally, never sent to Sednium servers

## How It Works
1. Users add their own API keys from supported providers
2. Users select a chat mode (Quick, Thinking, or Coding)
3. Users can add custom system instructions
4. Users chat with AI models through a unified interface
5. Conversations are processed directly through the chosen AI provider

## Technology
- Frontend: React with Tailwind CSS
- AI Orchestration: Sednium Rosette API
- Local AI: Ollama, LM Studio
- Model Hub: Hugging Face
