# Oorty - Project Specification

## Overview
Oorty is an advanced AI chat and coding orchestration platform built to harness multiple large language models seamlessly. This document serves as a comprehensive reference for AI models generating the `oorty.sednium.com` website.

## URLs
- **Hosted Application:** [oorty.sednium.com](https://oorty.sednium.com)
- **Official Publisher:** [sednium.com](https://sednium.com)

## Visual Identity & Color Scheme
The application utilizes a distinctive, elegant, and warm editorial visual identity.
- **Font Family:** `Source Serif 4` (serif)
- **Primary Background Color (sedYellow):** `#FDFBF7` (Milk White)
- **Primary Text & Accent Color (sedRed):** `#EC5E27` (Orange)

*Note for AI builder:* Ensure the website uses these colors prominently to maintain brand consistency. Backgrounds should lean heavily on `sedYellow` (Milk White) and text/interactive elements on `sedRed` (Orange).

## App Features

### 1. Multi Model Orchestration chat and coding using Sednium Rosette Api key (NEW)
- **Description:** A powerful new orchestration feature allowing seamless integration and dynamic routing of chat and coding tasks using the proprietary Sednium Rosette API key.

### 2. Multi-Provider & Model Support (Working)
- **Description:** Connect to numerous LLM providers using their respective API keys.
- **Supported Providers:** 
  - Google Gemini
  - OpenAI
  - Anthropic
  - XAI
  - Groq
  - OpenRouter
  - NVIDIA NIM
  - Local/Custom endpoints (e.g., Ollama)
  - Browser-native models

### 3. Specialized Chat Modes (Working)
- **Quick Mode:** Maximal speed, precision, and brevity.
- **Thinking Mode:** Deep, rigorous chain-of-thought analytical reasoning.
- **Coding Mode:** Elite software architect mode, writing clean, robust, and highly optimized code.

### 4. Custom System Instructions (Working)
- **Description:** Users can append additional custom system instructions on top of the selected Chat Mode to finely tune the AI's behavior.

### 5. Configuration Presets (Working)
- **Description:** Save and manage favorite configurations (Model Provider + Specific Model + Chat Mode + System Instructions) as easily accessible presets for rapid switching.

### 6. Attachments & Context (Working)
- **Description:** Add files and images into the chat context. 

### 7. Chat History Management (Working)
- **Description:** Comprehensive local chat history drawer. Features include viewing past conversations, pinning important chats, renaming conversations, and deleting old ones.

### 8. Theme and UI (Working)
- **Description:** Highly polished, responsive React interface styled with Tailwind CSS, leveraging custom animations (fade-in, slide-up, pop-up) and a custom color palette.

## Planned / Future Features

To ensure Oorty remains a state-of-the-art AI orchestration platform, the following features are planned for future releases:

### 1. Web Browsing & Search Grounding
- **Description:** Live internet access for models to fetch real-time data, summarize web pages, and ground responses in up-to-date information.

### 2. Document Processing & RAG (Retrieval-Augmented Generation)
- **Description:** Native support for parsing large PDF, Word, and Excel files. Users can upload massive knowledge bases and "chat" with their documents.

### 3. Voice & Audio Interaction (TTS/STT)
- **Description:** Seamless speech-to-text for hands-free prompting, and natural voice text-to-speech for reading model responses aloud.

### 4. Cloud Sync & Cross-Device History
- **Description:** Secure, encrypted cloud synchronization allowing users to access their chat history, presets, and preferences across desktop and mobile devices.

### 5. Advanced Agentic Tools
- **Description:** Empower the AI to execute code snippets natively, run API requests, and autonomously perform multi-step workflows directly within the chat interface.

### 6. Chat Export & Sharing
- **Description:** Generate public links for specific conversation threads or export chats cleanly to PDF, Markdown, or Notion.

### 7. Prompt & Preset Marketplace
- **Description:** A community hub where users can discover, share, and import highly optimized system prompts and model configurations.

### 8. Advanced Conversation Organization
- **Description:** Introducing folders, tags, and intelligent auto-categorization to help users manage growing chat histories effectively.
